## Express Starter Pack
### Open the terminal 
Enter the following command one by one!!

```
git clone "https://github.com/sifytul/express-starterpack.git" <repo name>
cd <repo name>
yarn
yarn dev
git remote remove origin
```
## Windows powershell
```
Remove-Item -Recurse -Force .git
```

## Linux or Mac
```
rm -rf .git/
```
